# AI Dev Navigator

一个面向初学者的 AI 开发工具信息集成仓库。
目标是把分散信息整理成一个可检索入口：**官方仓库、官网文档、安装方式、命名消歧、skills 生态、上手路径**。

## 你能在这里找到什么
- AI Coding CLI 工具索引（官方优先）
- 一键安装与常见报错排查
- Agent Skills / MCP 生态导航
- 初学者 7 天上手路径

## 内容结构
- `docs/tools-index.md`: 工具总表（仓库/官网/安装/备注）
- `docs/install-and-troubleshooting.md`: 新手安装排错手册
- `docs/skills-ecosystem.md`: skills 与 MCP 资源聚合
- `SOURCES.md`: 来源与核验时间

## 快速开始
1. 看 `docs/tools-index.md` 选一个 CLI。
2. 按 `docs/install-and-troubleshooting.md` 跑通安装。
3. 按 `docs/skills-ecosystem.md` 接入 skills/MCP。

## 声明
- 本仓库是信息集成与检索导航，不是官方评测排行。
- 对于命名冲突工具，会显式标注“官方/社区/待核验”。
