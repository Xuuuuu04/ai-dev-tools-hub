## Beginner Path (7 Days)
1. 选一个 CLI（Claude Code / Gemini CLI / Codex / Qwen Code）。
2. 用它完成一个最小改动并提交。
3. 增加项目 `AGENTS.md` 或规则文件。
4. 接入 1 个 MCP server。
5. 跑一次自动化 PR/Review 工作流。
6. 记录一次失败案例与修复过程。
7. 发布一个可复用 skills 包。

## Star-oriented Tips
- 明确受众：例如“后端团队专用 skills”。
- 输出可复制：命令、示例、预期结果。
- 保持更新：每周更新日志。
- 写清边界：适用与不适用场景。
